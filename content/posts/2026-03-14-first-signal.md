## Why Intentional Motion Matters

Motion should answer a user question. If the animation does not teach, guide, or delight with purpose, it is decoration.

## Build For Performance First

Start with a 60fps budget. Every effect must justify CPU and memory cost.

### Practical Rules

- Prefer transform and opacity over layout thrashing.
- Guard high-cost animation by media queries and reduced-motion.
- Reuse objects when rendering particles.

## Closing Thought

Great interfaces feel alive, but never loud.
